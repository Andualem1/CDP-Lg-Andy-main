# CSO Learning Hub Landing Page — Creative Brief

## 1. Purpose of this file

This file defines the creative, content, visual, and strategic direction for the improved public landing page of the CSO Learning Hub.

The purpose is to prepare the landing page design before Codex implementation. This file guides the generation of the golden mockup, the final `DESIGN.md`, the HTML reference, and the eventual `/` route implementation.

---

## 2. Screen identity

**Screen name:**  
Public Landing Page

**Route:**  
`/`

**Experience zone:**  
Public / learner-facing

**Primary users:**  
- Local and grassroots CSO staff
- CSO focal persons
- Programme participants
- DEC / WHH / partner stakeholders
- Potential course participants
- Reviewers and donors viewing the platform for credibility

---

## 3. Core strategic goal

The landing page must immediately communicate that the CSO Learning Hub is a credible, practical, locally grounded digital learning platform for strengthening local and grassroots CSO capacity.

The page must not feel like a generic LMS, a file repository, a software demo, or a donor-facing dashboard.

It should feel like:

> A professional digital learning space where local CSO staff can access practical courses, apply learning to real work, track progress, complete assessments, and earn certificates.

---

## 4. First-impression goal

Within the first 5 seconds, a visitor should understand:

1. This is a learning platform for CSOs.
2. It is practical and locally relevant.
3. It supports structured digital courses.
4. Participants can learn step by step, track progress, and earn certificates.
5. The platform is credible enough for DEC, WHH, CSF+ partners, and local CSO users.

---

## 5. Visual continuity rule

The landing page must preserve the photographic background-image direction from the earlier prototype and uploaded reference screenshots.

The hero and key trust-building sections should use photographic imagery showing:

- local CSO learning;
- group discussion;
- training or workshop settings;
- planning and proposal work;
- collaborative reflection;
- Ethiopian or locally grounded organizational learning contexts.

The photographic image direction is a continuity anchor.

Do not replace this with:

- abstract SaaS illustrations;
- generic Western corporate stock imagery;
- cartoon-style e-learning graphics;
- decorative images that do not feel locally relevant;
- dashboard-only mockups without human context.

---

## 6. What to preserve from the earlier prototype direction

Preserve:

1. Strong photographic hero background.
2. Local CSO collaboration / learning atmosphere.
3. Editorial headline feeling.
4. Calm, premium, institutional tone.
5. Course showcase section.
6. Feature cards.
7. Strong blue CTA section.
8. Dark institutional footer with partner recognition.
9. Human-centered and locally grounded feeling.

2. Typography hierarchy.
3. Navigation clarity.
4. Hero message strength.
5. CTA clarity.
6. Section spacing and rhythm.
7. Course card consistency.
8. Feature-card clarity.

---

## 8. Core message

The landing page should communicate:

> The CSO Learning Hub helps local and grassroots CSO staff access practical, structured, and accessible digital learning so they can strengthen skills, apply learning in real organizational work, track progress, and earn certificates.
