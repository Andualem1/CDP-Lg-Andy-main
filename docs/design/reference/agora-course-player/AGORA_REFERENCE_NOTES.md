# Agora Course Player Reference Notes

## 1. Purpose

This folder contains Agora course catalogue, course detail, and course-player screenshots registered as visual references for the CSO Learning Hub learner course player redesign.

Use these files for layout inspiration only. Do not copy Agora/UNICEF branding, logos, colors, or exact UI content.

## 3. Useful patterns

The strongest patterns to adapt are:

```txt
- institutional top header with learner utilities;
- hamburger-triggered global menu;
- catalogue and course detail structures that feel like a formal LMS;
- course launch page with clear objectives, audience, length, provider, and start action;
- persistent left course outline in the player;
- lesson sections with strong title banners;
- vertical stacked learning content;
- illustrations, callouts, quote/persona panels, resource lists, quizzes, result states, and Continue buttons;
- completion indicators in the course outline.
```

## 4. CSO Learning Hub adaptation

The CSO Learning Hub implementation must adapt these patterns to:

```txt
- DEC Blue #3B99D4;
- DEC Green #91C852;
- Deep Navy #0F172A;
- Light Background #F9FAFB;
- white cards;
- soft borders;
- rounded cards;
- accessible contrast;
- CSO capacity-building language and content.
```

## 5. Priority for implementation

For learner course-player layout, the Agora references and the local redesign spec are primary. Articulate 360 references remain useful for block-level interaction inspiration, but not for the overall player shell layout.
